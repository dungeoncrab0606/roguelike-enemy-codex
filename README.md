# Roguelike Enemy Codex

A data-driven TypeScript toolkit for generating deterministic enemy profiles for roguelike and RPG prototypes.

This project is an early-stage open-source extraction target from a real solo roguelike/RPG prototype. The goal is to move enemy data out of game-specific source files and into external JSON files that can be validated, version-controlled, and reused across AI-assisted game development workflows.

## Project status

Pre-alpha / planning repository.

The first implementation target is `v0.1.0`, focused on:

- external JSON enemy definitions
- JSON Schema validation
- deterministic generation from a seed
- enemy family and role modifiers
- rank-based stat tables
- threat scaling and spawn variance
- sample enemies for roguelike/RPG prototypes
